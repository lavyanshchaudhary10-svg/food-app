# 🍽️ Khana Khalo Ji

**Food delivery web app** — browse dishes, manage a cart, place orders, and track them. Built with a **React** frontend and an **Express + MongoDB** backend.

> *Khana Khalo Ji - Fixed and working* · Full-stack MERN-style food ordering experience.

---

## ✨ Features

| Area | What you get |
|------|----------------|
| **Landing** | Branded intro with stats and “Order Karo Abhi” CTA |
| **Menu** | Food cards by category (Burgers, Pizza, Pasta, Chinese, South Indian, Drinks, Desserts) with ratings, veg/non-veg, tags & images |
| **Wishlist** | Save favourites (stored in `localStorage`) |
| **Cart** | Add items, quantity updates, checkout flow |
| **Auth** | Signup / login with **JWT** + bcrypt-hashed passwords |
| **Orders** | Place order, view **My Orders**, status updates (Placed → Delivered) |
| **Profile & address** | User profile and delivery address pages |
| **UX** | Toasts, food detail modal, responsive UI (Poppins) |

---

## 🛠️ Tech Stack

| Layer | Stack |
|-------|--------|
| **Frontend** | React 18, React Router DOM 6, Create React App |
| **Backend** | Node.js, Express 4 |
| **Database** | MongoDB (Mongoose 7) |
| **Auth** | JWT (`jsonwebtoken`), `bcryptjs` |
| **Other** | CORS, dotenv |
